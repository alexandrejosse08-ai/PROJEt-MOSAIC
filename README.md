# PROJEt-MOSAIC
Le projet MOSAÏC (Morphology in Oral Rare Syndromes and Artificial Intelligence for Clinical Diagnosis) vise à développer des outils d'aide au diagnostic morphométrique des anomalies bucco-dentaires associées aux maladies rares de l'os et du cartilage, en combinant morphométrie géométrique 3D et apprentissage automatique.
Ce dépôt contient les scripts Python développés dans le cadre d'une étude préliminaire du projet MOSAÏC portant sur la discrimination de la morphologie palatine tridimensionnelle entre individus sains et individus atteints de dents incluses, à partir de 239 landmarks positionnés sur la vue occlusale du palais de 90 spécimens. Trois stratégies de prétraitement GPA sont comparées sur 50 partitions répétées afin d'évaluer l'impact de la contamination Procrustes sur les performances de classification. La stratégie optimale, GPA Algorithm 2, est ensuite utilisée pour comparer huit algorithmes de classification par validation croisée stratifiée 5-fold répétée 10 fois, les différences de performances étant testées statistiquement par le test de Friedman suivi d'un post-hoc de Nemenyi avec correction FDR. Les scripts produisent l'ensemble des figures et tableaux de résultats incluant les courbes ROC, la matrice des différences de F1-score et la heatmap des p-values corrigées.
Dépendances : Python 3.9+, scikit-learn, xgboost, numpy, pandas, matplotlib, seaborn, scikit-posthocs, statsmodels
